# 🛍️ 电商主图快速生成器 (抖音/小红书适配)

纯前端电商主图生成工具，自动抠图 + 模板编辑 + 一键导出。

## ✨ 功能特性

- 📸 **多方式上传** - 支持文件上传、拖拽、剪贴板粘贴
- 🎯 **自动抠图** - 基于 `@imgly/background-removal` 纯前端 WASM 自动移除背景
- 📱 **平台适配** - 一键切换 抖音 1:1 (1024×1024) / 小红书 3:4 (900×1200)
- 🎨 **5个预设模板** - 简约/渐变/纹理/深色/撞色多种风格
- ✏️ **灵活编辑** - 商品支持拖拽移动、滚轮缩放；文字双击编辑，可改大小颜色
- 💾 **高清导出** - 一键导出 2x 分辨率 PNG 图片

## 🛠️ 技术栈

- React + TypeScript + Vite
- Konva.js - 画布渲染
- TailwindCSS - 样式
- @imgly/background-removal - 前端背景移除 (WASM)

## 🚀 本地启动

```bash
# 克隆项目
git clone <你的仓库地址>
cd ecommerce-main-image-generator

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📖 使用说明

1. 上传商品图片（或拖拽/粘贴）
2. 等待自动移除背景
3. 选择平台（抖音/小红书）
4. 选择喜欢的模板
5. 调整商品位置和大小
6. 双击编辑标题文字，在右侧面板修改文字属性
7. 点击下载导出 2x PNG

## 📝 输出尺寸

- 抖音 1:1 - `2048 × 2048` px (2x)
- 小红书 3:4 - `1800 × 2400` px (2x)

## 🔒 隐私说明

所有处理都在浏览器本地完成，图片不会上传到任何服务器。
